# カレーレシピ生成アプリ
## 概要
- 上部のドロップダウンメニューから、用いる食材を選択
- Sendボタンを押して、選択した食材を用いたカレーのレシピを生成

## 使用方法
- .envファイルを作成し、　```REACT_APP_GOOGLE_API_KEY={Gemini API Key}```　を追加 ( {Gemini API Key} には利用者が取得したAPI Keyを入れる)
- 外部ライブラリのインストール
  - ```npm install axios```
  - ```npm install react-markdown```
  - ```npm install react-select```
- (アプリ起動) npm start
